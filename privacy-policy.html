Kebijakan Privasi Aplikasi

📋 Pendahuluan

Selamat datang di Nama Buah Hati — aplikasi kumpulan nama bayi Islami Indonesia. Kebijakan Privasi ini menjelaskan bagaimana kami menangani informasi pengguna saat menggunakan aplikasi kami dengan paket aplikasi com.namababyhati.nama_buah_hati.

✅ Aplikasi ini beroperasi secara offline. Anda dapat menggunakannya tanpa koneksi internet untuk menelusuri nama-nama bayi Islami.

🔒 Data Pribadi Tidak Dikumpulkan

Kami tidak mengumpulkan, menyimpan, atau memproses data pribadi pengguna dalam bentuk apapun. Secara spesifik:

  • Kami tidak meminta nama, alamat, atau informasi identitas Anda
  • Kami tidak mengumpulkan nomor telepon atau alamat email
  • Kami tidak melacak lokasi Anda
  • Kami tidak menyimpan riwayat pencarian ke server kami
  • Kami tidak memiliki sistem akun pengguna (registrasi/login)

👤 Akun Pengguna

Aplikasi Nama Buah Hati tidak mewajibkan maupun menyediakan fitur pendaftaran akun. Anda dapat langsung menggunakan seluruh fitur aplikasi tanpa perlu membuat akun atau masuk (login) dengan akun manapun.

📢 Iklan (Google AdMob)

Aplikasi ini menampilkan iklan yang disediakan oleh Google AdMob. Layanan iklan pihak ketiga ini dapat mengumpulkan dan memproses data tertentu sesuai kebijakan privasi mereka sendiri, antara lain:

  • Identifikasi perangkat (Device ID / Advertising ID)
  • Data penggunaan aplikasi untuk personalisasi iklan
  • Informasi jaringan dan lokasi perkiraan

Untuk informasi lebih lanjut mengenai cara Google mengumpulkan dan menggunakan data, silakan baca Kebijakan Privasi Google di: https://policies.google.com/privacy

ℹ️ Pengumpulan data oleh iklan dilakukan oleh Google, bukan oleh pengembang aplikasi Nama Buah Hati.

🛡️ Keamanan Data

Karena aplikasi berjalan secara offline dan tidak mengumpulkan data pribadi Anda, risiko terhadap keamanan data pribadi dari sisi kami sangat minimal. Seluruh konten nama bayi tersimpan secara lokal di perangkat Anda dan tidak pernah dikirim ke server manapun.

🔄 Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini sewaktu-waktu. Perubahan akan diberitahukan melalui pembaruan aplikasi di Google Play Store. Kami menyarankan Anda untuk meninjau halaman ini secara berkala. Kebijakan ini berlaku sejak dipublikasikan.
